Steps in building an Image

  • Pick an Image (careful of the file extension in the URL)
  • Record URL, cite it, comment for date accessm and record original dimensions
  • Use blockquote and comment as appropiate
  • Use img-tag with atributes src, alt, title, and style with width-height
